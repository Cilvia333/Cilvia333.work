# cilvia333.work

## About

This is cilvia333's portfolio website.

## installation

- node >= 12.18.3
- Yarn >= 1.22.4
- gatsby-cli >= 2.12.68

### cloning

```bash
git clone https://github.com/Cilvia333/Cilvia333.work.git
git checkout v3.0
yarn
```

### dev

```bash
yarn dev
```

### build

```bash
yarn build
```

### lint/format

```bash
yarn lint
yarm fmt
```
